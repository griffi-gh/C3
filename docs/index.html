C3 CPU Architecure

View old docs   GitHub

Info

C3 (CPU3) is a simple 16-bit processor architecture.

Registers

C3 has 4 (16 bit) general-purpose registers: A, B, C and D;
2 binary flag registers (Zero and Carry);
and 1 I/O register: IO_OUT (Write-only)

I/O

C3 can have up to 255 I/O ports.
Connector pins:
Name Type Description
IO_CLK Output Clock
IO_TRIGGER Output Send a pulse to the I/O device
IO_OUT Output Value of the IO_OUT register
IO_WAKE Input Unhalt the CPU
IO_IN Input Value for the IO_READ instr
